一.需求目的
(1)项目架构决定需要一个权限的模板控制台
(2)需要内部员工管理系统及员工销售记录系统
(3)需要销售统计系统
(4)需要虚拟消耗产品后台管理系统
二.个人负责的模块
(1)储备客户
    添加 储备客户 功能点 百度地图获取坐标 提交用户个人信息 添加跟进信息,修改跟进信息,转到试用客户,及权限问题,注销客户,客户信息修改
(2)正式客户 
    直接添加正式客户 功能点 百度地图获取坐标 客户充值 ,注销客户,添加欠费,  合同续约,登录日志,跟进,客户信息修改,单个权限分配
(3)字典管理
     字典是功能模板公司下发的功能性字典,所有功能里面的下拉菜单内容.
(4)版本管理
   版本外壳 具体版本 创建版本的弹窗及修改版本.
(5)财务流失
    财务流水分为 以成交客户里的 客户充值,添加欠费,以及直接添加欠费.
    添加财务流水 流水照片 查看财务流水 流水照片 偏好设置
